# DCANP Group App - versión Vercel + Supabase

Este proyecto usa el `index.html` que ya tenías y lo conecta a una API propia en Vercel.

## Qué incluye
- Frontend web listo
- API `/api/gas` compatible con `google.script.run`
- SQL completo para Supabase
- Inicializador para crear el primer administrador
- PDFs básicos de guías
- Links públicos de seguimiento

## Archivos principales
- `public/index.html`
- `public/setup.html`
- `api/gas.js`
- `api/setup.js`
- `api/guide-single.js`
- `api/guide-bulk.js`
- `api/report-link.js`
- `database.sql`
- `.env.example`
